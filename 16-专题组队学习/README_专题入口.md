## 专题关系

地瓜机器人新版教程桌宠侧重硬件入口、MagicBox 控制、语音、按钮、灯效和舵机交互。OpenClaw 家庭物资助手在这个基础上继续扩展到家庭物资管理、飞书通知、多维表格同步和前端展示。

如果读者第一次接触这套内容，建议先学习地瓜机器人新版教程桌宠，再学习 OpenClaw 家庭物资助手。

AMD ROCm 策略复刻专题侧重异构 GPU 环境中的策略训练、评估和排障，适合已经具备 LeRobot / MuJoCo 基础，并希望在 AMD 设备上复刻 ACT、SmolVLA 和 pi_0 的读者。它可以和第 06 章策略抓取教程配套学习。

## 资料整理建议

学习这些专题时，建议把教程正文、轻量脚本、指标表格和代表截图放在章节目录里。模型权重、缓存、下载包、镜像文件、`.env`、Hugging Face token、SSH 密钥和本机私有配置应放在自己的实验目录或缓存目录中，不要混进教程材料。
